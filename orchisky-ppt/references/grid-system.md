# Grid System · 画布坐标体系

> 可插拔模块。本文件是所有版式坐标的唐一来源。

---

## 画布基准

```
画布尺寸：960 × 540 px
可用内容区：左 40 / 右 40 / 上 64 / 下 48
内容宽度：880 px
内容高度：428 px
列网格：880 / 12 = 73.3px列，间距 16px
```

---

## Action Title 区域

```
左竖线： x=40, y=48, height=48
竖线宽度： 3px
竖线颜色： var(--brand-primary)

Action Title 文字：
  x = 52 (竖线右 9px)
  y = 48 (顶部对齐)
  height = 48
  垂直居中： y + height/2 = 基线
  font-size: 22px
  font-weight: Bold
  font-family: 标题字体
  fill: var(--brand-near-black)
```

---

## 内容区坐标

```
内容区起点： x=40, y=112 (全高 - 内边距上 - Action Title)
内容区宽： 880px
内容区高： 380px (540 - 112 - 48)

内容可主要分區：
  - 单列： x=40, width=880
  - 左半： x=40, width=424
  - 右半： x=496, width=424
  - 三等分： width=272, 间距=32
  - 四等分： width=196, 间距=24
```

---

## Chrome 区域

```
底部栏： x=0, y=492, width=960, height=48
底部栏颜色： var(--brand-primary)

页码区域： x=856, y=508, 字体 12px

公司Logo区域： x=40, y=504, 字体 14px
公司名字用白色
```
