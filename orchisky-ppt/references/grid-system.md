# Grid System · 画布坐标体系

> 可插拔模块。本文件是所有版式坐标的唯一来源。
> 画布统一为 **1280 × 720 px**，与所有 svg-skeleton-*.md 文件对齐。

---

## 画布基准

```
画布尺寸：1280 × 720 px
可用内容区：左右各 40px，上 100px，下 30px
内容宽度：1200 px（x=40 ~ x=1240）
内容高度：590 px（y=100 ~ y=690）
列网格：1200 / 12 = 100px列，间距 16px
```

---

## Chrome 区域（每页固定层）

```
顶部品牌蓝线   x=0    y=0    w=1280  h=3
页眉区         x=40   y=3    ~ y=30
页眉分割线     y=30
Action Title区  x=40   y=36   h=58（竖线 3px + 文字）
AT 分割线      y=94
内容区         x=40   y=100  w=1200  h=590
页脚分割线     y=690
页脚区         x=40   y=690  ~ y=720
```

---

## Action Title 区域

```
左竖线：  x=40, y=36, width=3, height=52
竖线颜色：var(--brand-primary)

Action Title 文字（单行）：
  x=52（竖线右 9px）
  y=70（垂直居中于 36~88）
  font-size: 22px
  font-weight: Bold
  font-family: 标题字体
  fill: var(--brand-near-black)

Action Title 文字（双行）：
  行1 y=54，行2 y=78，font-size: 20px
```

## 内容区坐标速查

```
内容区起点：  x=40, y=100
内容区结束：  x=1240, y=690
内容宽：      1200px
内容高：      590px

分区参考：
  单列全宽：    x=40,  w=1200
  左半：        x=40,  w=590
  右半：        x=646, w=594
  左 60%：      x=40,  w=720
  右 40%：      x=776, w=464
  三等分：      w=380, 间距=16（x=40/436/832）
  四等分：      w=277, 间距=16（x=40/333/626/919）
  五等分：      w=216, 间距=15

HERO 布局（顶部全宽+底部分栏）：
  顶区：  x=40,  y=100, w=1200, h=255
  底区起：y=371（顶区下 + 间距16）
  底区高：h=309（690-371-10）
```
