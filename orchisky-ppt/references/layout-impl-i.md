# I 系 · 震撼类（Impact）

**共同特征**：极高视觉冲击力，信息密度低（10–40%），以单一强信息驱动受众注意力。
**使用时机**：每部分开场、核心论点页、数字证据页、情绪转折页。

---

## I-01 · 单数字超大屏

**版式意图**：用 100px+ 数字占满页面，一眼定输赢。

```
画布：1280 × 720
大数字区：x=140, y=100–420（字号180px，底对齐 y=380）
断线：  x1=140, y1=410, x2=420, y2=410（1.2px）
断言文字：x=140, y=456（30px Bold 宋体）
支撑文字1：x=140, y=506（14px 灰色）
支撑文字2：x=140, y=530（14px 灰色）
警告文字（可选）：x=140, y=610（13px 危险红，letter-spacing=1）
```

**占位符说明**：
- `[big_number]`：核心数字，含单位符号（如 87%、¥3.2亿、×4.6）
- `[assertion]`：断言句，≤15字（如"远超行业均值"）
- `[support_1/2]`：补充说明，各≤20字
- `[warning]`：可选，红色警示语（如"窗口期仅剩90天"）

**信息密度**：15%

## I-02 · 一句话宣言（深色底）

**版式意图**：深色背景打破节奏，白字宣言制造记忆点。无 Chrome 框架。

```
背景：全页 fill=#0A1F3A（深海蓝）
章节标记：x=80, y=80（11px 灰蓝，letter-spacing=3）
宣言行1：x=120, y=300（42px Bold 宋体 白色）
宣言行2：x=120, y=360（42px Bold 宋体 白色）
副语行1：x=120, y=450（28px 浅蓝灰 #B8C8D8）
副语行2：x=120, y=490（28px 浅蓝灰 #B8C8D8）
来源：  x=1200, y=640 text-anchor=end（13px 灰蓝）
```

**约束**：
- 宣言每行 ≤14 字，两行合计 ≤24 字
- 副语行可省略（留白即留白）
- 无页眉 / Action Title / 页脚（此版式自成一页）

**信息密度**：15%

## I-03 · 反差双栏（现状 vs 目标）

**版式意图**：左红右绿，视觉落差强化问题→机会的叙事。

```
画布内容区：x=40, y=100, w=1200, h=590
左栏区：    x=100–620（左侧60%约）
  标签：  x=100, y=200（11px 危险红 letter-spacing=2）
  大字：  x=100, y=310（80px Bold 宋体 危险红）
  描述：  x=100, y=370（16px 深灰）
竖分割线：  x=640, y1=160, y2=580（0.5px 灰色）
箭头符号：  x=628, y=388（24px 灰色 →）
右栏区：    x=720–1240（右侧）
  标签：  x=720, y=200（11px 成功绿 letter-spacing=2）
  大字：  x=720, y=310（80px Bold 宋体 成功绿）
  描述：  x=720, y=370（16px 深灰）
```

**占位符说明**：
- `[left_label]`：现状标签（如"TODAY · 现状"）
- `[left_big]`：左侧大字（如 "23%" 或 "87天"）
- `[right_label]`：目标标签（如"TARGET · 目标"）
- `[right_big]`：右侧大字（如 "60%" 或 "30天"）

**信息密度**：25%

## I-04 · 数据冲击（大数字+细节支撑）

**版式意图**：左侧大数字震撼，右侧竖排3条理由说明为何这个数字重要。

```
左区：x=40–780
  大数字：x=140, y=380（170px Bold 宋体 品牌蓝）
  断线：  x1=140, y1=400, x2=440, y2=400（1.5px）
  断言：  x=140, y=440（28px Bold 宋体 近黑）
竖分割线：x=800, y1=140, y2=610（1px 灰色）
右区：x=860–1240（支撑列）
  标题：  x=860, y=170（13px 危险红 letter-spacing=2）
  红线：  x1=860, y1=192, x2=1000, y2=192（1.8px）
  理由1：标题 y=234 / 正文 y=258（14px Bold + 13px 灰）
  分割线：y=290（0.5px）
  理由2：标题 y=332 / 正文 y=356
  分割线：y=388（0.5px）
  理由3：标题 y=430 / 正文 y=454
```

**占位符说明**：
- `[big_number]`：核心数字
- `[assertion]`：≤12字断言
- `[right_header]`：支撑列标题（如"因为以下三点"）
- `[reason1/2/3_label]`：各≤10字
- `[reason1/2/3_desc]`：各≤20字

**信息密度**：35%

## I-05 · 时间窗警告

**版式意图**：时间轴可视化 + 当前位置红点 + 窗口期大字，传递紧迫感。

```
时间轴：x1=100, y1=200, x2=1180, y2=200（1.5px 浅灰）
过去节点：cx=200, cy=200, r=8（灰填充）
  标签：x=200, y=234 text-anchor=middle（12px 灰）
当前节点：cx=640, cy=200, r=20（危险红填充）
  内标签：x=640, y=207 text-anchor=middle（13px Bold 白）
  外标签：x=640, y=250 text-anchor=middle（13px Bold 危险红）
未来节点：cx=1080, cy=200, r=8（灰色虚线描边）
  标签：x=1080, y=234（12px 灰）
警告主标题：x=640, y=360 text-anchor=middle（34px Bold 宋体 近黑）
警告副文字：x=640, y=404 text-anchor=middle（15px 灰）
窗口期大字：x=640, y=510 text-anchor=middle（60px Bold 宋体 危险红）
窗口期标签：x=640, y=558 text-anchor=middle（14px 灰）
```

**占位符说明**：
- `[past_label]`：过去时间点（如"2023 Q1"）
- `[now_label]`：当前标签（如"NOW"）
- `[now_tag]`：当前状态（如"决策窗口"）
- `[future_label]`：未来节点（如"2026 Q3"）
- `[warning_title]`：主警告（如"市场窗口即将关闭"）
- `[window_period]`：窗口期（如"90天"）
- `[window_label]`：窗口说明（如"剩余可用窗口期"）

**信息密度**：30%
