# Phase 2 · 材料解析与信息结构化

> 可插拔模块。独立升级不影响其他Phase。
> 本Phase将原始输入转化为结构化信息清单，是Phase 3的原材料。

---

## 入场条件

- Phase 1完成卡已确认
- BRAND_RULES已加载
- 受众卡已产出

## 执行步骤

### 步骤1：材料收集

如果用户有材料，读取并解析。
如果没有材料，提醒：「没有材料会影响内容质量，建议至少提供关键数据或主要观点。」
然后继续引导用户口述。

### 步骤2：信息四类标注

对所有信息打标签：
- **[事实]** — 可验证的客观陈述
- **[数据]** — 有具体数字的陈述
- **[观点]** — 判断、结论、建议
- **[空缺]** — 需要但缺失的信息

### 步骤3：输出信息清单

```
信息清单：
[事实] ...
[数据] ...
[观点] ...
[空缺] ...
```

对[空缺]项，提出处理建议：补充、推断、标注待确认、或删除。
