# Layout Library · 版式库

> 本文件被 Phase 5（版式规划）和 Phase 7（逐页设计）调用。
> 版式按视觉主角类型分为 A-G 七类，每类包含1个推荐版式和2个备选版式。

---

## A类 · 数字聚焦型

### A-01 · 全屏震撞数字（推荐）
```
布局：数字占全屏60%+，一行注释，极简留白
适用：单一核心指标、ROI数字、里程碑数据
```

### A-02 · 数字左+解释右（备选1）
```
布局：左侧60%大数字，右侧40%文字注解
适用：需要对数字做补充说明的场景
```

### A-03 · 三指标并列（备选2）
```
布局：三个KPI横排，各占1/3
适用：需要同时展示3个相关指标
```

---

## B类 · 对比张力型

### B-01 · 现状vs目标（推荐）
```
布局：左右各半，左灰右蓝，中间虚线
适用：现状与理想状态的对比
```

### B-02 · 问题→解法（备选1）
```
布局：左红右绿，中间→箭头
适用：问题与解决方案的映射
```

### B-03 · 方案选项比较（备选2）
```
布局：2-4个选项上下堆叠，推荐选项高亮
适用：多方案评估，需要指出推荐方向
```

---

## C类 · 流程叙事型

### C-01 · 横向步骤流（推荐）
```
布局：3-5步横排，箭头连接，关键步骤高亮
适用：实施计划、执行步骤、流程说明
```

### C-02 · 时间轴纵向（备选1）
```
布局：左侧垂直轴，右侧事件描述
适用：历史发展、项目里程碑、时间线展示
```

### C-03 · 漏斗/箭头流程（备选2）
```
布局：3个阶段箭头形状横排
适用：转化漏斗、价值传递链、因果链条
```

---

## D类 · 数据图表型

### D-01 · 图表全幅主体（推荐）
```
布局：图表占页面70%+，标题+注释贴边
禁用：饼图、三维图表、渐变填充
```

### D-02 · 图表左+洞察右（备选1）
```
布局：图表占左侧60%，右侧关键数字+2-3行解读
适用：图表有明确结论，需引导受众看到重点
```

### D-03 · 主图左+辅图右（备选2）
```
布局：主图左侧，右上次要图+右下数据表格
适用：需要多维度展示同一议题
```

---

## E类 · 矩阵框架型

### E-01 · 2×2优先级矩阵（推荐）
```
布局：两轴四象限，圆圈代表选项，高价值区高亮
适用：优先级决策、风险/收益分析、战略选择
```

### E-02 · 多方案评分列（备选1）
```
布局：4列对比，每列一个方案，推荐列高亮
适用：方案比选、供应商评估、工具选型
```

### E-03 · 价值/复杂度矩阵（备选2）
```
布局：2×2矩阵，高价值低复杂度象限高亮
适用：功能优先级、项目排期、改进机会评估
```

---

## F类 · 引言震撞型

### F-01 · 结论引言全屏（推荐）
```
布局：大字居中，极简留白
适用：章节起首、关键结论页、震撞性判断
```

### F-02 · 章节分隔页（备选1）
```
布局：左侧品牌色竖条，章节号+标题+2-3行摘要
适用：多章节PPT的章节切换
```

### F-03 · 引用语录页（备选2）
```
布局：大引号，核心引用文字，出处小字
适用：专家观点、客户反馈、权威数据引用
```

---

## G类 · 结构分解型

### G-01 · 树状分解图（推荐）
```
布局：顶层节点居中，向下展开3个分支
适用：MECE框架展示、问题分解、价值链分析
```

### G-02 · 三支柱并立（备选1）
```
布局：三等分竖列，各支柱含标题+2-3个子要点
适用：三大支撑、三个价值维度、三阶段策略
```

### G-03 · 层级金字塔（备选2）
```
布局：梯形层叠，顶层高亮最重要
适用：战略/管理/执行三层、能力模型、需求层级
```
