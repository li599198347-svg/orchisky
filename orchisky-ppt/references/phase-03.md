# Phase 3 · 核心论点提炼 ★

> 可插拔模块。独立升级不影响其他Phase。
> 本Phase是整个流程唯一的"思想飞跃"环节。

---

## 入场条件

- Phase 2完成卡已确认
- 信息清单已产出
- 受众卡可访问

## 执行步骤

### 步骤1：论点提炼

基于信息清单和受众卡，生成候选灵魂句（≤30字）：

```
灵魂句公式：
「[受众]面临[核心问题]，
  [解决方案/方向]能实现[具体价值]」
```

提供2-3个候选，让用户选择或修改。

### 步骤2：确认

用户确认后，锁定灵魂句。后续所有Phase的内容都服务于这一句话。

### 步骤3：支撑论点

生成3-5个支撑灵魂句的论点，每个论点一句话。
