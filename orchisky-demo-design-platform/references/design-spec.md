# 原型设计规范 — Orchisky 平台版 V1.0

> 版本 V1.0 · 发布 2026-04-19 · 基础 Ant Design 5 · 主色 #1677FF
>
> 适用范围：客户项目基于 **Orchisky 平台** 进行二次开发时，所有 AI 生成的原型、方案 Demo、概念稿。
> 并列文档：《原型设计规范 V1.0（品牌版）》用于独立品牌 Demo；《品牌与设计规范 V2.0》是兰之天整体品牌源头。

## 0. 为什么有这份规范

Orchisky 平台本身已经基于 **Ant Design 5** 做了深度定制。客户看到的真实系统长什么样,规范的全部意义就是**让我们的 Demo 长一模一样**。

让 AI 长出"平台感"的方式不是"画得好看",而是**严格遵循 Ant Design 5 的默认值**:色值、圆角、间距、字重、组件结构,一个都不要改。

本规范的所有数值均来自真实平台截图的像素级实测,不是推测。

## 1. 平台视觉 DNA（10 条铁律）

| # | 铁律 | 关键值 |
|---|---|---|
| 01 | 基于 Ant Design 5 | 所有 token、组件、圆角、间距继承 Ant 5 默认值 |
| 02 | 主色 | `#1677FF`（Ant `colorPrimary`），禁改 |
| 03 | 顶栏 | 高 48px，深色 `#001529`，白字，左置 Logo，激活 tab **白底蓝字** |
| 04 | 侧栏 | **白底**（非彩色），宽 200–240px，激活项 `#E6F4FF` 药丸 + 蓝字 |
| 05 | 圆角 | 默认 `6px`（Ant 5 标准），小控件 4px，大容器 8px |
| 06 | 按钮 | 高 32px，主色实填，次按钮白底灰边，危险按钮红填或红描边 |
| 07 | 表格 | 表头 `#FAFAFA`，主键列蓝色链接，工具栏右上放批操作 |
| 08 | 表单 | **多列网格（3–4 列）**，必填红 `*` 在标签前，标签右对齐加冒号 |
| 09 | 状态 | 用**实色填充小药丸**（Ant Tag），非淡底圆点 |
| 10 | 字体 | Noto Sans SC / PingFang SC，数字/ID 用 JetBrains Mono |

违反任何一条 = 不再是平台风格。

## 2. 色彩系统（Ant Design 5 原生）

### 2.1 主色与语义色（禁改）

| Token | Hex | 用途 |
|---|---|---|
| `colorPrimary` | `#1677FF` | 主按钮、链接、激活态、主键列 |
| `colorPrimaryHover` | `#4096FF` | 主按钮 hover |
| `colorPrimaryActive` | `#0958D9` | 主按钮按下 |
| `colorSuccess` | `#52C41A` | 成功实填、Switch-on |
| `colorWarning` | `#FAAD14` | 警告、待审核 |
| `colorError` | `#FF4D4F` | 错误、危险按钮 |
| `colorInfo` | `#1677FF` | 信息提示（同主色） |

| `red-1` | `#FFF2F0` | Error Alert、Tag 淡底 |

|---|---|---|
| gray-2 | `#FAFAFA` | 表头、页面背景 |

## 3. 字体与字号

| 用途 | 字体 |
|---|---|
| 中文 UI | `"Noto Sans SC", "PingFang SC", "Microsoft YaHei"` |
| 数字 / ID / 编码 | `"JetBrains Mono", "SF Mono", monospace` |

**强制**：所有工单号、物料编码、金额、数量、百分比、日期时间必须用等宽字体。

---

## 4. 间距 · 圆角 · 控件高度

| 圆角 Token | px | 用途 |
|---|---|---|
| `borderRadiusSM` | 4 | Tag、Badge |
| `borderRadius` | 6 | 按钮、输入框（默认） |
| `borderRadiusLG` | 8 | 卡片、Modal |

| 控件高度 Token | px | 用途 |
|---|---|---|
| `controlHeightSM` | 24 | 小按钮 |
| `controlHeight` | 32 | 默认（最常用） |
| `controlHeightLG` | 40 | 大按钮 |

---

## 5. 布局骨架（顶栏 + 侧栏）

**顶栏**：高 48px，底色 `#001529`，Logo 左置，激活 Tab 白底蓝字，右侧图标区 5–7 个。

**侧栏**：宽 200–240px，白底，激活项 `#E6F4FF` 药丸 + `#1677FF` 字 + 500 字重。

**内容区**：背景 `#FAFAFA`，padding 16px，白卡 + 6px 圆角 + 1px `#F0F0F0` 边（不用 box-shadow）。

---

## 6. 按钮

| 类型 | 样式 |
|---|---|
| Primary | 实填 `#1677FF`，白字 |
| Default | 白底，边 `#D9D9D9` |
| Danger（实填） | 底 `#FF4D4F`，白字 |
| Danger（描边） | 白底，边 `#FFCCC7`，字 `#FF4D4F` |

## 7. 表格

| 元素 | 规格 |
|---|---|
| 表头 | `#FAFAFA` · 13px · 600 · padding 12px 16px |
| 数据行 | 高 44px · 下边 1px `#F0F0F0` |
| Hover 行 | `#F5F5F5` |
| 主键列 | `#1677FF` + 等宽字体，hover 下划线 |
| 数字列 | **右对齐** + 等宽字体 + 千分位 |

---

## 8. 表单（新增 / 编辑页）

- 列数：3 列（1440px） / **4 列**（宽屏>1600px）
- 标签：宽 100–120px，**右对齐**，加冒号
- 必填：红 `*` 在标签**前**，`#FF4D4F`
- 输入框：高 32px，圆角 6px
- 段标题：600 · 15px · `#262626`，**无图标无装饰**

---

## 9. 标签与状态语义

**实色填充**（用于业务状态）：`[已审核蓝] [已同步绿] [已驳回红] [冻结灰]`

**淡底描边**（用于分类标记）：`[采购单] [成品] [半成品]`

## 10. AI-slop 禁用清单（平台版）

| # | 错误模式 | 应该怎样 |
|---|---|---|
| 01 | 自创主色（#1070F0） | 只用 `#1677FF` |
| 02 | 圆角 12px / 16px | 按钮 6 · Tag 4 · Modal 8 |
| 03 | 卡片加阴影 | 1px `#F0F0F0` 描边 |
| 04 | 侧栏彩色 | 白底 + `#E6F4FF` 药丸 |
| 05 | 彩色表头 | `#FAFAFA` 灰 |
| 06 | 标签上浮（Material） | 标签左置 + 右对齐 |
| 07 | 状态淡底 + 圆点 | 实色填充小药丸 |
| 08 | Emoji / 3D 图标 | @ant-design/icons 线性 |
| 09 | 假数据离谱 | 参照业务假数据字典 |
