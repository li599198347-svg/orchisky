# 原型设计规范 — Orchisky 平台版 V1.0

> 版本 V1.0 · 发布 2026-04-19 · 基础 Ant Design 5 · 主色 #1677FF
>
> 适用范围：客户项目基于 **Orchisky 平台** 进行二次开发时，所有 AI 生成的原型、方案 Demo、概念稿。

---

## 0. 为什么有这份规范

Orchisky 平台本身已经基于 **Ant Design 5** 做了深度定制。让 AI 长出"平台感"的方式不是"画得好看"，而是**严格遵循 Ant Design 5 的默认值**。

---

## 1. 平台视觉 DNA（10 条铁律）

| # | 铁律 | 关键值 |
|---|---|---|
| 01 | 基于 Ant Design 5 | 所有 token、组件、圆角、间距继承 Ant 5 默认值 |
| 02 | 主色 | `#1677FF`，禁改 |
| 03 | 顶栏 | 高 48px，深色 `#001529`，白字，激活 tab **白底蓝字** |
| 04 | 侧栏 | **白底**，宽 200–240px，激活项 `#E6F4FF` 药丸 + 蓝字 |
| 05 | 圆角 | 默认 `6px`，小控件 4px，大容器 8px |
| 06 | 按钮 | 高 32px，主色实填，次按钮白底灰边 |
| 07 | 表格 | 表头 `#FAFAFA`，主键列蓝色链接，工具栏右上批操作 |
| 08 | 表单 | **多列网格(3–4 列)**，必填红 `*` 在标签前，标签右对齐加冒号 |
| 09 | 状态 | 用**实色填充小药丸**（Ant Tag） |
| 10 | 字体 | Noto Sans SC / PingFang SC，数字/ID 用 JetBrains Mono |

## 2. 色彩系统

### 主色与语义色

| Token | Hex | 用途 |
|---|---|---|
| `colorPrimary` | `#1677FF` | 主按钮、链接、激活态 |
| `colorSuccess` | `#52C41A` | 成功 |
| `colorWarning` | `#FAAD14` | 警告 |
| `colorError` | `#FF4D4F` | 错误、危险 |

### 浅色背景

| Token | Hex | 用途 |
|---|---|---|
| `blue-1` | `#E6F4FF` | 侧栏激活 |
| `green-1` | `#F6FFED` | Success Alert |
| `gold-1` | `#FFFBE6` | Warning Alert |
| `red-1` | `#FFF2F0` | Error Alert |

### 中性灰阶

| Token | Hex | 用途 |
|---|---|---|
| gray-2 | `#FAFAFA` | 表头、页面背景 |
| gray-4 | `#F0F0F0` | 分隔线 |
| gray-5 | `#D9D9D9` | 输入框边框 |
| dark-menu | `#001529` | 顶栏深色底 |

### 禁改清单 ⛔

- ❌ 把主色改成 `#1070F0`（那是品牌版色）
- ❌ 给侧栏加彩色或渐变
- ❌ 给表头加背景色

## 3. 字体与字号

| 用途 | 字体 |
|---|---|
| 中文 UI | `"Noto Sans SC", "PingFang SC", "Microsoft YaHei"` |
| 数字 / ID / 编码 | `"JetBrains Mono", monospace` |

字号阶梯（Ant 5）：

| px | 用途 |
|---|---|
| 14 | 正文、表格（默认） |
| 16 | 大号正文 / 段落标题 |
| 12 | 辅助、时间戳 |

---

## 4. 间距 · 圆角 · 控件高度

```
间距：4 / 8 / 12 / 16 / 24 / 32 / 48px
圆角：Tag 4px / 按钮&输入框 6px / 卡片&Modal 8px
控件高度：SM=24 / 默认=32 / LG=40px
```

> ⚠️ 平台版默认 6px 圆角，品牌版是 2/4px，**不要混用**。

---

## 5. 布局骨架

### 顶栏
- 高度 48px，底色 `#001529`
- 激活 Tab：白底蓝字
- 右侧图标区 36×36px

### 侧栏
- 宽 200–240px，底色 `#FFFFFF`
- 激活项：底 `#E6F4FF` + 字 `#1677FF` + 字重 500

### 内容区
- 背景 `#FAFAFA`
- 卡片：白底 + 6px 圆角 + 1px `#F0F0F0` 边（**不用 box-shadow**）

---

## 6. 按钮

| 类型 | 样式 |
|---|---|
| Primary | 实填 `#1677FF`，白字 |
| Default | 白底，边 `#D9D9D9` |
| Danger（实填） | 底 `#FF4D4F`，白字 |
| Danger（描边） | 白底，边+字 `#FF4D4F` |

尺寸：LG=40 / 默认=32 / SM=24px

---

## 7. 表格

| 元素 | 规格 |
|---|---|
| 表头 | `#FAFAFA` · 13px · 600 |
| 数据行 | 高 44px · 14px |
| Hover | `#F5F5F5` |
| 主键列 | `#1677FF` + 等宽字体 |
| 数字列 | **右对齐** + 等宽字体 + 千分位 |

禁用：斑马纹、彩色表头、行阴影、数字列左对齐 ⛔

---

## 8. 表单

- 3–4 列网格，横距 24px，纵距 16px
- 标签宽 100–120px，**右对齐**，加冒号
- 必填红 `*` 在标签**前**
- 段标题：600 · 15px · **无图标无装饰**

禁用：标签上浮、必填号放标签后、圆角 12px、段标题加蓝色竖线 ⛔

---

## 9. 标签与状态

### 实色填充（业务状态） ⭐ 平台标志
```
[已审核蓝] [已同步绿] [已驳回红] [冻结灰]
```

### 淡底描边（分类标记）
```
[采购单] [成品] [紧急] [VMI]
```

状态颜色：蓝=进行中/待处理，绿=完成，红=驳回/异常，橙=警告，灰=禁用/归档

## 10. 典型页面模板

### LIST 列表页
```

```


## 11. AI-slop 禁用清单

| 错误模式 | 应该怎样 |
|---|---|
| 自创主色（紫蓝渐变/#1070F0） | 只用 `#1677FF` |
| 圆角 12px/16px | 按钮6·Tag4·Modal8 |
| 卡片加阴影 | 1px `#F0F0F0` 描边 |
| 侧栏彩色 | 白底+`#E6F4FF`药丸 |
| 彩色表头 | `#FAFAFA` 灰 |
| 标签上浮 | 标签左置+右对齐 |
| 状态淡底+圆点 | 实色填充小药丸 |
| 段标题加蓝色竖线 | 加粗黑字直接写 |
