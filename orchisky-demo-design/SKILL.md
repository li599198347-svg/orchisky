# Orchisky Demo Design 技能

## 概述

本技能帮助你生成严格符合兰之天 Orchisky 原型规范的 B 端 Demo 界面。

**三种场景，九个子模板**：
- **A · 管理后台**（1440×900）：A.2 MES 排产看板 / A.3 WMS 库龄分析 / A.4 通用后台
- **B · 移动端 App**（375×812）：B.2 工单执行 / B.3 数据查询 / B.4 审批
- **C · 大屏可视化**（1920×1080）：C.2 生产监控 / C.3 园区监控 / C.4 运营驾驶舱

---

## 执行流程（每次必须严格按此顺序）

### Step 1：读取规范

**强制**：生成任何原型前，必须先读取以下两个文件：

```
references/design-spec.md      ← 原型设计规范全文（视觉铁律 + 组件规格 + 数据字典）
references/prompt-templates.md ← Prompt 模板库（骨架结构 + 字段 + 数据要求）
```

### Step 2：识别场景

根据用户需求，判断对应的场景类型：

| 用户提到 | 选用模板 |
|---------|----------|
| 工单、排产、MES、生产管理 | A.2 MES 排产看板 |
| 库存、库龄、WMS、仓储 | A.3 WMS 库龄分析 |
| 客户管理、订单、通用后台 | A.4 通用后台 |
| 一线工人、现场、手机报工 | B.2 工单执行 |
| 移动端看日报、数据查询 | B.3 数据查询 |
| 审批、OA、移动审批 | B.4 审批 |
| 大屏、生产监控、指挥中心 | C.2 生产监控 |
| 园区、仓储监控中心 | C.3 园区监控 |
| 集团驾驶舱、运营总览 | C.4 运营驾驶舱 |

如果用户描述模糊，**直接问一个问题**：「是管理后台、移动端还是大屏？」不要问多个。

### Step 3：生成 HTML

严格按照对应模板的骨架、组件规格、数据要求生成完整 HTML。

**输出格式**：
- 完整的单文件 HTML + 内联 CSS
- 不依赖任何 UI 框架（Bootstrap / Ant Design 等全部禁止）
- 不引用任何外部 CDN（除 Google Fonts 的字体和 Lucide Icons CDN）
- 文件保存到 `/mnt/user-data/outputs/` 并用 `present_files` 呈现给用户

## 视觉铁律（每次生成必须全部遵守，违反任意一条 = 不合格）

| # | 铁律 | 关键值 |
|---|------|--------|
| 01 | 主色唯一 | `#1070F0`，禁第二装饰色 |
| 02 | 中文字体 | `"Noto Sans SC", "PingFang SC", "Microsoft YaHei"` |
| 03 | 数字字体 | `JetBrains Mono`（ID / 工单号 / 金额 / 编码） |
| 04 | 8pt 栅格 | 间距只能是 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 px |
| 05 | 圆角 | 0 / 2 / 4 px，移动端最多 8 px |
| 06 | 分隔 | 1px 实线 `#E2E2E2`，禁 box-shadow |
| 07 | 图标 | Lucide 线性，描边 1.5px，只用 Lucide |
| 08 | 信息密度 | 默认 14px / 行高 40px，表格单屏 15+ 行 |

---

## AI-slop 禁用清单（生成前自查，出现任意一条 = 重做）

1. 紫蓝粉渐变卡片 → 改为白底 + 1px 灰边
2. 毛玻璃 backdrop-filter → 改为不透明白底
3. Emoji / 3D 图标 / 拟物图标 → 改为 Lucide 线性图标
4. 饼图 / 环形图 / 3D 图表 → 改为柱状图 / 折线图 / 数字
5. 假数据离谱（张三 / $999,999 / test） → 按假数据字典
6. 所有卡片 + 大圆角 + 阴影「SaaS 风」→ 直角 / 2px 圆角 + 1px 描边
7. 斑马纹表格 → 悬停蓝底 `#EAF2FE`
8. 营销腔文案（高效 / 智能 / 赋能）→ 删除
9. 动画超过 150ms → 裁剪
10. Lorem ipsum / 英文占位 → 全中文

## 关键组件规格速查

**按钮**：高 32px / 圆角 2px / 主按钮 `#1070F0` 底白字（同屏仅 1 个）/ 次按钮白底灰边

**表格**：行高 40px / 表头 `#F5F5F5` / 悬停 `#EAF2FE` / 禁斑马纹 / 数字列右对齐 JetBrains Mono

**统计卡**：上方 12px 灰色指标名 / 下方 32px JetBrains Mono 数字 / 白底 1px 灰边 / 圆角 4px

**状态徽标**：6px 圆点 + 文字 + 淡色背景 + 同色描边（四件套齐全）
- 成功绿 `#1A7A42` / 信息蓝 `#1070F0` / 警告橙 `#9A5200` / 危险红 `#B01C1C` / 中性灰 `#555`

**侧栏**：220px / 白底 / 当前项左侧 3px 蓝条 + 蓝字 / 组标题 12px 灰色

**顶栏**：56px / 白底 / 顶部 3px `#1070F0` 线

**弹窗**：宽度 480/640/800 / 遮罩 `rgba(10,14,20,.45)` / 圆角 ≤ 4px

```css

主文字: #FFFFFF
次要文字: rgba(255,255,255,.6)
强调数字: #6FA3F5（品牌亮蓝）
异常: #FF7070
```

## 交互要求

- 可点元素：`cursor: pointer` + 悬停色变化
- 不可点占位：`cursor: default`，**禁止**加 hover 效果
- 动画上限：150ms，超过一律删除
