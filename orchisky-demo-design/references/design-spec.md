## 为什么需要这份规范

兰之天经常需要用 AI 快速生成原型图、Demo 界面，用于给客户演示方案。过去的问题：

- 不同人、不同 AI 工具输出的 Demo 视觉千差万别，「不像一家公司的事情」
- 最基础的排版、组件、假数据规则每次都要重新说明，浪费时间
- AI 容易出现「AI slop」特征：彩色渐变、花里胡哨、假数据离谱

本规范解决两个可执行性问题：
1. **AI 一次就能生成合格 Demo**（视觉 DNA + Prompt 模板 + 组件规格）
2. **不同人、不同 AI 用同一规范输出 = 可互认**（骨架、字典、禁用清单固化）

# 01 · 视觉 DNA · 8 条铁律

这 8 条是「一眼识别是不是兰之天出品」的硬指标，AI Prompt 里必须原样照抄。

| 编号 | 铁律 | 关键值 |
|------|------|--------|
| 01 | **主色唯一** | `#1070F0` 一个色，禁第二装饰色 |
| 02 | **中文字体** | `"Noto Sans SC", "PingFang SC", "Microsoft YaHei"` |
| 03 | **数字字体** | `JetBrains Mono`（所有 ID / 工单号 / 金额） |
| 04 | **8 pt 栅格** | 间距只能是 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 |
| 05 | **圆角** | 0 / 2 / 4 px 三档，移动端允许 8 px |
| 06 | **分隔** | 1 px 实线 `#E2E2E2`，不用 `box-shadow` |
| 07 | **图标** | Lucide 线性，描边 1.5 px，只用一个图标库 |
| 08 | **信息密度** | 默认 14 px / 行高 40 px，单屏 15+ 条 |

# 02 · 三种场景骨架

## 2.1 Web 后台管理系统（最常用）

**画布尺寸**：1440 × 900，响应式向 1280 下限

**版面骨架**（顺序不可调整）：
```
┌────────────────────────────────────────────┐
│ 顶栏 56 px（3 px 蓝顶线 + Logo + 导航）     │
├──────┬─────────────────────────────────────┤
│ 侧栏  │ 面包屑                              │
│ 220  │ 页面标题 + 操作按钮                  │
│      │ 统计卡（一排 4-5 个）               │
│      │ 筛选器                              │
│      │ 表格 / 看板 / 图表                  │
└──────┴─────────────────────────────────────┘
```

**关键尺寸**：
- 顶栏高度 56 px，白底，顶部 3 px 品牌蓝线
- 侧栏宽度 220 px（可折叠至 60 px）
- 内容区 padding 16 px，背景 `#FAFAFA`
- 禁止深色顶栏 / 彩色侧栏 / Logo 居中

## 2.2 移动端 App

**画布尺寸**：375 × 812 px（iPhone X 基准）

**三区结构**：
- 状态栏 28 px（iOS）
- 顶部导航 44 px
- 内容区 flex 1（背景 `#FAFAFA`）
- 底部 Tab 56 px（iOS 含 Home Indicator 84 px）

**与 Web 差异**：
- 允许 8 px 圆角
- 允许轻阴影 `0 2px 8px rgba(0,0,0,.04)`
- 字号 ≥ 13 px，点击区域 ≥ 44 px
- 其余一律继承 V2.0

## 2.3 大屏可视化

**画布尺寸**：1920 × 1080（16:9 固定）

**布局**：顶部 44 px 标题栏 + 3 × 3 九宫格，中间 2 × 1 为主图表位

**调色**：
- 背景深蓝近黑 `#02081C`（禁渐变 / 星空 / 粒子）
- 面板：6% 透明蓝填充 + 30% 品牌蓝描边
- 文字：白 `#FFFFFF` / 弱化白 60% / 品牌亮蓝 `#6FA3F5` / 异常 `#FF7070`
- 主标题 28 px，关键数字 40-60 px（JetBrains Mono 700）

# 03 · B 端十件套组件库

## 3.1 按钮 Button

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| Primary | `#1070F0` 底 + 白字 | 主操作，同屏仅 1 个 |
| Default | 白底 + `#CCC` 边 + 深灰字 | 次操作 |
| Danger | 白底 + 深灰边 + `#B01C1C` 字 | 删除类 |
| Text | 无边框 + 蓝字 | 第三级操作 |

- 标准高度 32 px · 小尺寸 24 px
- padding 0 14 px · 字号 13 px · 圆角 2 px

## 3.2 表格 Table

- 行高 **40 px**
- 表头灰底 `#F5F5F5`，字号 12 px
- 禁用斑马纹，只用 1 px 实线分隔
- 悬停 / 选中蓝底 `#EAF2FE`
- 数字列右对齐，用 JetBrains Mono
- 操作列固定 100-120 px 宽，链接式文字按钮

## 3.3 表单 Form

- 标签右对齐，宽度 120 px
- 控件高度 32 px
- 必填项 `*` 红色 `#B01C1C`，放字段前
- 行间距 12 px
- 禁止下划线样式输入框

## 3.4 审批流程节点

- 已完成：蓝底白勾
- 当前：白底蓝描边 + `0 0 0 3px rgba(16,112,240,.15)` 蓝光晕
- 未到达：灰描边白底
- 节点间 2 px 实线连接

## 3.5 看板 Kanban

- 4 列结构（待 / 中 / 检 / 完）
- 列头底部 2 px 色条区分
- 卡片左侧 2 px 蓝条
- 卡片间距 8 px

## 3.6 状态徽标 Badge

**四件套，缺一不可**：6 px 圆点 + 文字 + 淡色背景 + 同色描边

| 语义 | 颜色 | 使用 |
|------|------|------|
| 成功 | `#1A7A42` | 已完成 / 通过 |
| 信息 | `#1070F0` | 在产 / 处理中 |
| 警告 | `#9A5200` | 待审 / 预警 |
| 危险 | `#B01C1C` | 异常 / 失败 |
| 中性 | `#555` | 已归档 / 禁用 |

**禁止仅靠颜色传达状态**（色觉障碍用户需要文字标识）。

## 3.7 空 / 加载 / 错误态

三态统一结构：**图标 → 主标题 → 说明文字 → 操作按钮**

## 3.8 面包屑 + 页面标题

- 面包屑字号 12 px，灰色 `#888`，分隔符 `/`
- 页面标题 20 px 加粗 `#1A1A1A`
- 主操作按钮右对齐

## 3.9 弹窗 Modal

- 宽度三档：480 / 640 / 800
- 遮罩 `rgba(10,14,20,.45)`
- 禁止圆角 > 4 px / 禁止缩放动画

## 3.10 分页 Pagination

- 按钮高度 24 px
- 当前页主蓝底白字
- 左侧显示总数和当前范围

# 04 · 假数据字典

| 字段 | 格式示例 | 规则 |
|------|----------|------|
| 人名 | 王建国 · 李文静 · 张卫东 | 长三角姓氏 + 70-80 后常用名。禁 张三/李四/test |
| 公司名 | 吴江精密机械 · 无锡华瑞电机 | 地名 + 行业描述。禁 阿里/华为/某某科技 |
| 工单号 | WO-20260412-0038 | 前缀 + 日期 + 4 位流水。禁 12345/TEST-001 |
| 物料编码 | M-BRG-A2024-02 | 类别缩写 + 产品代号 + 版本，10-16 字符 |
| 金额 | ¥ 1,285,400.00 | 千分位 + 2 位小数，万-百万区间 |
| 数量 | 1,280 件 | 带千分位 + 单位 |
| 百分比 | 87.2% | 保留 1 位小数，避免假整数 100%/50% |
| 日期时间 | 2026-04-12 14:32:08 | 列宽紧张用 04-12 14:32，禁混用格式 |
| 手机号 | 138 **** 2847 | 中间 4 位脱敏。禁真实号码 |
| 设备编码 | CNC-02-05 · 总装-A 线 | 可读性强，带类型前缀 |

# 05 · 图表使用规范

推荐：折线图 / 柱状图 / 堆叠柱图 / 仪表盘 / 进度条 / 热力图

禁用：**饼图** / **环形图** / **3D 图（任何类型）**

图表三不用：不用多彩渐变配色 / 不用图表动画 / 不用离散图例（数据直接标注）

---

# 06 · 图标系统

- **只用一个图标库**（推荐 Lucide Icons）
- 描边 1.5 px，尺寸 16 / 20 / 24 px 三档
- 禁用：Emoji / 3D 渲染 / 多彩图标 / 线性和填充混用

---

# 07 · 状态与反馈语言

反馈黄金原则：成功态不打断用户 / 错误态必须解释原因 / 警告态必须给出下一步操作

---

# 08 · 可点击范围标注规则

- 可点元素：`cursor: pointer` + 悬停色变化
- 禁用 / 占位：不变色 + `cursor: default`
- 不能点的元素**禁止**加 `pointer` 或 `hover`

---

# 09 · AI-slop 禁用清单

| # | 特征 | 替代 |
|---|------|------|
| 01 | 紫蓝粉渐变卡片 | 白底 + 1 px 灰边 |
| 02 | Glassmorphism 毛玻璃 | 不透明白底 |
| 03 | Emoji + 3D 图标 | Lucide 线性图标 |
| 04 | 饼图 / 3D 柱图 | 柱状图 / 折线图 / 数字 |
| 05 | 假数据离谱（张三 / $999,999） | 按假数据字典 |
| 06 | 所有卡片圆角 + 阴影 | 直角 / 2 px 圆角 + 1 px 描边 |
| 07 | 斑马纹表格 | hover 行 `#EAF2FE` |
| 08 | 营销腔（高效 / 智能 / 赋能） | B 端用信息，不用口号 |
| 09 | 过度动画（数字滚动 / 卡片浮动） | 最多 150 ms 基础过渡 |
| 10 | Lorem ipsum / 英文占位符 | 全中文界面 |
