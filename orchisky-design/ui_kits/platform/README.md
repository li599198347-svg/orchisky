# Platform UI Kit (Orchisky 平台版 · Ant 5)

Covers the **Orchisky Platform** — customer-facing apps built on Ant Design 5 with Orchisky presets.
Primary color: `#1677FF` (Ant 5 colorPrimary). Radius up to 8px.

## Files

| File | Description |
|------|-------------|
| `index.html` | Interactive admin console (Dashboard / 工单列表 / 工单详情) |
| `AppHeader.jsx` | Dark top bar (`#001529`, Ant dark menu) |
| `SideNav.jsx` | 220px side navigation (white, blue active pill) |
| `Dashboard.jsx` | KPI row + recent orders table |
| `OrderList.jsx` | Searchable, paginated order table |
| `OrderDetail.jsx` | Detail view with form + audit log |

Sources: `references/原型设计规范-Orchisky平台版-V1.0.md`
