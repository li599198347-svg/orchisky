# Field PDA UI Kit (Orchisky Field V1.0 · 现场版)

Covers **手持 PDA + 工业 Pad** (factory-floor focused, inherits V2.0 brand tokens).

## Components

| File | Component | Canvas |
|------|-----------|--------|
| `LoginScreen.jsx` | PDA login page | 480×800 |
| `PDAFrame.jsx` | Shell with StatusBar + Topbar + Body + BottomCTA | 480×800 |
| `ScanScreen.jsx` | Home — scan-entry menu | 480×800 |
| `ReceiveScreen.jsx` | WMS inbound order list | 480×800 |
| `ReceiveDetailScreen.jsx` | WMS inbound order detail + form | 480×800 |
| `index.html` | All screens composed side-by-side | static HTML |

## Token reference

- Primary: `--blue-500 #1070F0`
- StatusBar bg: `--ink-800 #1A1A1A`
- Font UI: Source Han Sans SC / Microsoft YaHei
- Font numbers: Roboto Mono
- Border radius: 2 / 4 / 6 px (no capsule)
