# 原型设计规范 — Orchisky 平台版 V1.0

> 版本 V1.0 · 发布 2026-04-19 · 基础 Ant Design 5 · 主色 #1677FF
>
> 适用范围：客户项目基于 **Orchisky 平台** 进行二次开发时，所有 AI 生成的原型、方案 Demo、概念稿。
> 并列文档：《原型设计规范 V1.0（品牌版）》用于独立品牌 Demo；《品牌与设计规范 V2.0》是兰之天整体品牌源头。

---

## 0. 为什么有这份规范

Orchisky 平台本身已经基于 **Ant Design 5** 做了深度定制。客户看到的真实系统长什么样，规范的全部意义就是**让我们的 Demo 长一模一样**。

本规范的所有数值均来自真实平台截图的像素级实测，不是推测。

## 1. 平台视觉 DNA（10 条铁律）

| # | 铁律 | 关键值 |
|---|---|---|
| 01 | 基于 Ant Design 5 | 所有 token、组件、圆角、间距继承 Ant 5 默认值 |
| 02 | 主色 | `#1677FF`（Ant `colorPrimary`），禁改 |
| 03 | 顶栏 | 高 48px，深色 `#001529`，白字，左置 Logo，激活 tab **白底蓝字** |
| 04 | 侧栏 | **白底**（非彩色），宽 200–240px，激活项 `#E6F4FF` 药丸 + 蓝字 |
| 05 | 圆角 | 默认 `6px`（Ant 5 标准），小控件 4px，大容器 8px |
| 06 | 按钮 | 高 32px，主色实填，次按钮白底灰边，危险按钮红填或红描边 |
| 07 | 表格 | 表头 `#FAFAFA`，主键列蓝色链接，工具栏右上放批操作 |
| 08 | 表单 | **多列网格（3–4 列）**，必填红 `*` 在标签前，标签右对齐加冒号 |
| 09 | 状态 | 用**实色填充小药丸**（Ant Tag），非淡底圆点 |
| 10 | 字体 | Noto Sans SC / PingFang SC，数字/ID 用 JetBrains Mono |

违反任何一条 = 不再是平台风格。

## 2. 色彩系统（Ant Design 5 原生）

### 2.1 主色与语义色

| Token | Hex | 用途 |
|---|---|---|
| `colorPrimary` | `#1677FF` | 主按钮、链接、激活态 |
| `colorSuccess` | `#52C41A` | 成功状态 |
| `colorWarning` | `#FAAD14` | 警告 |
| `colorError` | `#FF4D4F` | 错误、危险按钮 |

### 2.2 中性灰阶

| Token | Hex | 用途 |
|---|---|---|
| gray-1 | `#FFFFFF` | 卡片底色 |
| gray-2 | `#FAFAFA` | 表头、页面背景 |
| gray-4 | `#F0F0F0` | 分隔线 |
| gray-5 | `#D9D9D9` | 输入框边框 |
| gray-10 | `#262626` | 标题主文字 |
| dark-menu | `#001529` | 顶栏深色底 |

### 2.3 禁改清单

- ❌ 把主色改成 `#1070F0`（那是品牌版色）
- ❌ 给侧栏加彩色（渐变蓝/深色均不可）
- ❌ 给表头加背景色

## 5. 布局骨架

### 顶栏（48px）

底色 `#001529`，高 48px，Logo 左置高 22-24px，一级 Tab Active 白底蓝字

### 侧栏（200-240px）

底色 `#FFFFFF`，激活项：`#E6F4FF` 底 + `#1677FF` 字 + 500 字重

### 内容区

底色 `#FAFAFA`，padding 16px，卡片白底 + 6px 圆角 + 1px `#F0F0F0` 边（**不用 box-shadow**）

## 6. 按钮

| 类型 | 样式 |
|---|---|
| Primary | `#1677FF` 实填，白字 |
| Default | 白底，边 `#D9D9D9`，字 `rgba(0,0,0,.88)` |
| Link | 透明底，字 `#1677FF` |
| Danger（实填） | 底 `#FF4D4F`，白字（不可逆操作） |
| Danger（描边） | 白底，边 `#FFCCC7`，字 `#FF4D4F`（工具栏）|

同屏主按钮不超过 1 个。禁渐变背景、外发光、圆角 > 8px。

## 7. 表格

| 元素 | 规格 |
|---|---|
| 表头 | `#FAFAFA` · 13px · 600 · padding 12px 16px |
| 数据行 | 高 44px · 14px · 下边 1px `#F0F0F0` |
| Hover 行 | `#F5F5F5` |
| 选中行 | `#E6F4FF` |
| 主键列 | `#1677FF` + 等宽字体 |
| 数字列 | 右对齐 + 等宽 + 千分位 |
| 操作列 | 固定右侧 80-120px，Link 按钮 |

**禁用**：斑马纹、彩色表头、圆角单元格、数字列左对齐。

---

## 8. 表单

- **列数**：3 列（1440px）/ **4 列**（宽屏>1600px）
- **标签**：宽 100-120px，**右对齐**，加冒号
- **必填**：红 `*` 在标签**前**，`#FF4D4F`
- 输入框：高 32px，圆角 6px，边 `#D9D9D9`
- 段标题：600 · 15px · `#262626`，**无图标无装饰**

**AI 最容易犯的错**：标签上浮（Material 风）、必填号放标签后、段标题加蓝色竖线。

---

## 9. 状态标签

**实色填充小药丸（平台标志式符号）**：

```
[已审核]–蓝  [已同步]–绿  [已驳回]–红  [未同步]–灰  [可用]–蓝  [冻结]–灰
```

| 业务含义 | 颜色 |
|---|---|
| 进行中/待处理 | 蓝 |
| 已完成/成功 | 绿 |
| 异常/失败/驳回 | 红 |
| 警告/临期 | 橙 |
| 中性/禁用 | 灰 |

## 10. 典型页面模板

### LIST 列表页

筛选卡片（3条件×N行）→ [提交][重置] → 工具栏（筛选+批操作+导出）→ 表格（带复选/分页）

### DETAIL 详情页

[← back][详情 tab] → 基础信息（分段）→ 关联子表 → [编辑][打印][导出]

### CREATE 新增/编辑页

[← back][添加 tab] → 基础信息（分段 3-4列）→ 明细子表 → [提交][取消]

## 11. AI-slop 禁用清单（平台版）

|---|---|---|

| 04 | 侧栏彩色 | 白底 + `#E6F4FF` 药丸 |
| 05 | 彩色表头 | `#FAFAFA` 灰 |
| 06 | 标签上浮 | 标签左置 + 右对齐 |
| 07 | 状态淡底圆点 | 实色填充小药丸 |
| 08 | 蓝色竖线段标题 | 加粗黑字直接写 |
