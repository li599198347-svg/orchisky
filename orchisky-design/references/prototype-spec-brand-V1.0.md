# 01 · 视觉 DNA · 8 条铁律

| # | 铁律 | 关键值 |
|------|------|--------|
| 01 | **主色唯一** | `#1070F0` 一个色，禁第二装饰色 |
| 02 | **中文字体** | `"Noto Sans SC", "PingFang SC", "Microsoft YaHei"` |
| 03 | **数字字体** | `JetBrains Mono`（所有 ID / 工单号 / 金额） |
| 04 | **8 pt 栅格** | 间距只能是 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 |
| 05 | **圆角** | 0 / 2 / 4 px 三档，移动端允许 8 px |
| 06 | **分隔** | 1 px 实线 `#E2E2E2`，不用 `box-shadow` |
| 07 | **图标** | Lucide 线性，描边 1.5 px，只用一个图标库 |
| 08 | **信息密度** | 默认 14 px / 行高 40 px，单屏 15+ 条 |

# 02 · 三种场景骨架

## Web 后台管理系统（1440×900）

```
┌────────────────────────────────────────────┐
│ 顶栏 56 px（3 px 蓝顶线 + Logo + 导航）     │
├──────┬─────────────────────────────────────┤
│ 侧栏  │ 面包屑                              │
│ 220  │ 统计卡（一排 4-5 个）               │
│      │ 筛选器                              │
│      │ 表格 / 看板 / 图表                  │
└──────┴─────────────────────────────────────┘
```

顶栏高 56px，白底 + 顶部 3px 品牌蓝线；侧栏 220px（可折叠 60px）；内容区 padding 16px，背景 `#FAFAFA`

## 移动端 App（375×812）

状态栏 28px + 顶部导航 44px + 内容区 flex 1 + 底部 Tab 56px
允许 8px 圆角 + 轻阴影；字号 ≥ 13px；点击区域 ≥ 44px

## 大屏可视化（1920×1080）

背景 `#02081C`，9宫格布局，面板 6% 透明蓝填充 + 30% 蓝描边
标题 28px，关键数字 40-60px JetBrains Mono 700

# 03 · B 端十件套组件库

**按钮** · Primary `#1070F0` 底 + 白字；标准 32px，小 24px；圆角 2px

**表格** · 行高 40px；表头灰底 `#F5F5F5`；1px 分隔，禁斑马纹；数字右对齐 JetBrains Mono

**表单** · 标签右对齐 120px；控件高 32px；必填 `*` 红色放字段前；行间距 12px

**KPI 卡** · 白底 + 1px `#E2E2E2` 边 + 顶部 2px 语义色条；数字 JetBrains Mono 600

**状态徽标** · 6px 圆点 + 文字 + 淡色背景 + 同色描边；必须颜色+符号双通道

**弹窗 Modal** · 480/640/800；遮罩 `rgba(10,14,20,.45)`；禁圆角 > 4px

**空/加载/错误态** · 56px 虚线图标框 + 标题 + 说明 + 操作按钮

**面包屑 + 页面标题** · 12px 灰色，20px 加粗标题，主操作按钮右对齐

**看板 Kanban** · 4 列；列头底部 2px 色条；卡片左侧 2px 蓝条；间距 8px

**分页** · 24px；当前页主蓝底白字；左侧显示总数

# 05 · 图表使用规范

**推荐**：折线图 / 柱状图 / 堆叠柱图 / 仪表盘 / 进度条 / 热力图

**禁用**：**饼图**、**环形图**、**3D 图（任何类型）**、多彩渐变配色、图表动画、独立图例

---

# 06 · 图标系统

- 只用 Lucide Icons，描边 1.5px
- 尺寸：16 / 20 / 24px 三档
- 颜色继承父元素，交互态用主蓝
- **禁用**：Emoji、3D/拟物、混用多库、Font Awesome

# 07 · 状态与反馈语言

| 状态 | 文案 | 视觉 |
|------|------|------|
| 成功 | 保存成功 · 已提交 | 绿色 Toast 3s 消失 |
| 处理中 | 进行中 · 待审核 | 蓝色徽标 |
| 警告 | 库存预警 · 即将到期 | 橙色徽标 + 操作按钮 |
| 错误 | 异常 · 已停机 | 红色徽标 + 原因简述 |
| 空数据 | 暂无数据 | 虚线图标 + 说明 + 按钮 |

# 09 · AI-slop 禁用清单

| # | 特征 | 替代 |
|---|------|------|
| 01 | 紫蓝粉渐变卡片 | 白底 + 1px 灰边 |
| 02 | Glassmorphism 毛玻璃 | 不透明白底 |
| 03 | Emoji + 3D 图标 | Lucide 线性图标 |
| 04 | 饼图/3D 图表 | 柱状图/折线图/数字 |
| 05 | 假数据离谱 | 按字典（第4章）|
| 06 | 所有卡片圆角+阴影 | 直角/2px + 1px 描边 |
| 07 | 图标线性填充混用 | 整屏只用一种风格 |
| 08 | 过度动画 | 最多 150ms 基础过渡 |
| 09 | 文案啰嗦营销腔 | B 端用信息，不用口号 |
| 10 | Lorem ipsum | 全中文界面 |
