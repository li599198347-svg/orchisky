# Orchisky 技能合集

Orchisky 品牌全系列 AI 技能，覆盖 Demo 原型设计、平台后台、品牌规范执行、PPT 制作四大场景。

## 技能目录

| 技能 | 说明 |
|------|------|
| [orchisky-demo-design](./orchisky-demo-design/) | 兰之天 Orchisky B端原型/Demo 界面设计（移动端/大屏/通用） |
| [orchisky-demo-design-platform](./orchisky-demo-design-platform/) | PC 后台管理系统原型（1440px，基于 Ant Design 5 定制） |
| [orchisky-design](./orchisky-design/) | 品牌视觉规范强制执行器，含完整品牌规范文档 |
| [orchisky-ppt](./orchisky-ppt/) | 全流程 PPT 制作，对标麦肯锡/德勤顶级咨询公司标准 |

## 使用方式

每个技能目录下有 `SKILL.md` 作为技能入口，`references/` 目录存放规范参考文档，`assets/` 目录存放工具脚本。
